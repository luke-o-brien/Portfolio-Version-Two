MyFoundSounds MERN Project

A group project which involved building a full stack react application which was a social media site based around audion. The site allows users to upload sounds from there device as well as search for and listen to other users sounds. For this project we created our own back end API using Mongo Db and Express as well as utilising the cloudinary api for file storage. To aid development of the project our group utilised Jira and scrum working to plan the project and assign tasks and track progress through the two one week sprints.
